# reactEMS

build this project with below tech.

`react`、`redux`、`postcss`.

now we use `express` in server side, and use `node-postgres` to connect PostgresSQL.

## Install

```
npm install
```

## Run

start server in development environment.

```
npm start
```

in production environment we should use:

```
npm run start:prod
```

have fun :)
